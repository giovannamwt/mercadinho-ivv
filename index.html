Conheça o mercadinho solidário

O mercadinho solidário é um projeto criado para ajudar as famílias da comunidade a garantirem alimentação de uma maneira digna, cada família recebe uma quantidade de verba para adquirir os alimentos que precisa, essa verba é disponibilizada a medida que a família se envolve no instituto através de cursos profissionalizantes ou reforço, levando a uma maior aderência da comunidade aos programas do instituto e fazendo com que as família não recebam uma simples cesta, mas os alimentos que realmente precisam.
